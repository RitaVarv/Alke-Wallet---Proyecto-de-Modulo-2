# Alke Wallet

## Descripción del Proyecto

Es una aplicacaión web que simula la gestion de operaciones básicas de una billetera virtual con un menú principal y con la opción de realizar simulación de transferencias y compras.

Se creo con el objetivo de poner en practica y aplicar conceptos fundamentales de desarrollo Front-End utilizando HTML, JavaScriop y CSS.

## Funcionalidades

-Inicio de sesión
-Menú principal de navegación
-SImulación de depósitos a la cuenta
-Simulación de transferencias
-Visualización de historial de movimientos 

## Tecnologías utilizadas

-HTML5
-CSS3
-JavaScript
-JQuery
-Bootstrap

## Justificación tecnica

-HTML dividido en páginas para facilitar la navegación
-CSS compartido para mantener consistencia visual
-JavaScript modularizado en archivos segun funcionalidad para no tener un archivo único muy extenso.
-Uso de `localStorage` para simular la mantención de datos.
